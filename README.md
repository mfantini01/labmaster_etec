# LabMaster ETEC 🧪

O **LabMaster ETEC** é um jogo educacional desenvolvido para auxiliar alunos do 1º ano do curso técnico em Química na identificação de materiais de laboratório e na compreensão de sistemas experimentais. O projeto é uma parceria entre o Instituto Mauá de Tecnologia e a ETEC Júlio de Mesquita.

## 🚀 Sobre o Projeto
O principal objetivo deste sistema é reduzir a curva de adaptação dos alunos ao ambiente laboratorial, tornando o aprendizado mais dinâmico e seguro. O game permite que os estudantes pratiquem a associação entre vidrarias, equipamentos e suas funções específicas em um ambiente digital.

## ✨ Funcionalidades
- **Quiz Educativo**: Sistema de múltipla escolha com imagens de alta qualidade para identificação de materiais.
- **Tipos de Questões**: Perguntas de associação (material → função, sistema experimental, etc.).
- **Progressão**: Níveis de dificuldade adaptáveis ao desempenho do aluno.
- **Ajuda Dinâmica**: Ferramentas como dicas e eliminação de alternativas incorretas.
- **Gestão Pedagógica**: Painel administrativo para professores cadastrarem, editarem ou excluírem questões.
- **Monitoramento**: Sistema de autenticação para alunos e professores, permitindo o acompanhamento individualizado de desempenho.

## 🛠 Tecnologias Utilizadas
O projeto utiliza uma arquitetura robusta e multiplataforma:

| Categoria | Tecnologia | Justificativa |
| :--- | :--- | :--- |
| **Linguagem** | Dart | Linguagem de tipagem estática, compilada nativamente. |
| **Framework** | Flutter | Criação de interface multiplataforma (Android, iOS, Windows). |
| **Banco de Dados** | SQLite | Funcionamento 100% offline via `sqflite`. |
| **Gerenc. de Estado** | Provider | Gestão de dados globais (usuário, pontuação). |
| **Segurança** | Crypto | Armazenamento seguro de senhas via hash. |

## 🎨 Identidade Visual
O projeto segue as diretrizes da **Identidade Visual do Centro Paula Souza**, garantindo uma interface intuitiva, com contraste adequado e acessível para estudantes do ensino médio.
